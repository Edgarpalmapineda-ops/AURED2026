# AURED2026

Pagina web redisenada de AURED Manteleria.

## Archivos

- `index.html`: pagina principal.
- `assets/`: imagenes, logotipos, favicon, mascota y fondo.
- `AURED2026.code-workspace`: archivo para abrir el proyecto en Visual Studio Code.

## Para editar

Abre `AURED2026.code-workspace` en Visual Studio Code y modifica `index.html`.

## Para publicar en Cloudflare Pages

Configuracion recomendada:

- Framework preset: `None`
- Build command: dejar vacio
- Build output directory: `/`

Esta pagina es estatica: HTML, CSS, JavaScript e imagenes.
